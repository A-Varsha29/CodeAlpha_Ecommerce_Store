# CodeAlpha Ecommerce Store
A robust, full-stack e-commerce web application built using the Django framework. This project demonstrates model-view-controller (MVC) architecture, clean database design, and dynamic front-end template rendering.

# Features
-> Product Catalog: Dynamic rendering of store products with modular HTML templates.
-> Robust Backend: Structured Django apps ('av_store' and 'core') managing data workflow.
-> Database Migrations: Fully configured relational database schema ready for production updates.
-> Repository Hygiene: Clean version control configuration utilizing optimized '.gitignore' boundaries.

# Tech Stack
-> Backend: Python, Django Framework
-> Frontend: HTML5, CSS3, Django Template Language (DTL)
-> Database: SQLite3 (Local development)
